# 贷款管理系统

基于 Vue 2 + Vant 2.x 的移动端贷款管理仪表板应用。

## 功能特性

- 📊 **重点指标展示**: 四个核心指标的实时数据展示
- 📈 **趋势图表**: 使用 ECharts 展示近30天数据趋势
- 🔄 **标签排序**: 支持拖拽排序，自定义标签顺序
- 📱 **移动优先**: 完全响应式设计，适配各种移动设备
- 🎨 **简洁UI**: 基于 Vant UI 组件库，界面简洁专业
- 📅 **日期筛选**: 支持日期选择和多维度数据筛选

## 技术栈

- Vue 2.6
- Vant 2.x (移动端UI组件库)
- ECharts 5.x (图表库)
- Sortable.js (拖拽排序)
- Vue Router (路由管理)

## 安装运行

\`\`\`bash
# 安装依赖
npm install

# 启动开发服务器
npm run serve

# 构建生产版本
npm run build
\`\`\`

## 主要功能

### 首页
- 四个可切换的标签页：贷款余额、贷款授信客户、贷款用信客户、贷款投放
- 每个标签显示当前值、日变化、月变化、年变化
- 近30天趋势折线图
- 支持标签拖拽排序，顺序保存到本地存储

### 详情页
- 数据筛选（产品类型、日期、分支机构）
- 详细的数据汇总
- 大尺寸趋势图表（支持滑动查看）
- 分支机构数据列表
- 贷款投放页面支持双折线对比图

## 数据说明

当前使用模拟数据（mockData.js），实际使用时可替换为 API 调用。
